Site Intent

Red Barn Garden Center is a premier landscape nursery in Austin, Tx that specializes in Texas Native Plants. The store features a wide selection of Texas Native plants that need to be seen on the site. With nearly three acres of land that Red Barn resides upon, it is difficult to know all that the store has to offer. The site needs to allow users to sign up for memberships and find the current sales the store is having.

Client Goals

Create a more intuitive design that allows the user to find what their looking fore more efficiently. Make a better layout that functions more practically. Come up with a new color scheme. Newer photos need to be added to be consistent with the new look. Better navigation system.

User Goals

Better understand the landscape business. Update the code to html 5 and correct any errors. Make more efficient and better looking nav system.

Competitive Analysis

There are a few other Landscape Nursery's in the immediate area that are the biggest competition. This creates stiff competition, so it is important to create a website that stands apart.
